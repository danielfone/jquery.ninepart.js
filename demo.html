Dynamic Nine-Part Images

View plain | View dynamic

Using a jQuery function, we can achieve complex borders on elements that maintains semantic markup and clean css. Concept is based on Cocoa's NSDrawNinePartImage. In this implementation however, we only use four images to save on page loading time. Styling should retain complex layouts.

Example code: show... 
<script type="text/javascript">
  jQuery(function($){
    // Load a nine part image around the p
    // tag with class box. Images loaded will
    // be 'bg-all.png', 'bg-mid.png', 'bg-sides.png'
    // and 'bg-ends.png'.
    // Border and margin sizes are derived from the
    // child element's CSS.
    $('p.box').ninePartImage('bg-%.png');
    // Any additional CSS can be specified like so.
    $('p.box').ninePartCSS({position: 'absolute'});
  });
</script>
