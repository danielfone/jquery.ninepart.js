Dynamic Nine-Part Images

Using a jQuery function, we can achieve complex borders on elements that maintains semantic markup and clean css. Concept is based on Cocoa's NSDrawNinePartImage. In this implementation however, we only use four images to save on page loading time. Styling should retain complex layouts.

Two identically marked up and styled lists are illustrated. The top <ul> has a class of 'border'.

Source code:

<script type="text/javascript">
  jQuery(function($){
    // First arg is the image filename template,
    // the second is the size of the border in pixels.
    $('ul.border').ninePartImage('bg-%.png', 8);
    $('ul.border li').ninePartImage('bg-%.png', 8);
  });
</script>
<style type="text/css">
  ul { list-style: none; clear: both; border: 8px solid blue; height: 66px}
  li { float: left; padding: 5px; margin: 10px; border: 8px solid green; }
</style>
