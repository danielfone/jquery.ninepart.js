Dynamic Nine-Part Images

Nine-part image styling retains complex layouts. Two identically marked up and styled lists are illustrated. The top <ul> has a class of 'border'.

Javascript used:

<script type="text/javascript">
  jQuery(function($){
    // First arg is the image filename template,
    // the second is the size of the border in pixels.
    $('ul.border').ninePartImage('border-%.png', 8);
    $('ul.border li').ninePartImage('border-%.png', 8);
  });
</script>
